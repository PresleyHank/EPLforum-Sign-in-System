# 
用于易语言论坛（http://bbs.eyuyan.com/）签到 [非精易论坛]。

部署说明：
1.打开腾讯云 登陆后在控制台选择云函数
2.选择函数服务 新建
3.选择 【自定义创建】并且在地域中选择你常驻地址附近的。例：广州
4.运行环境设置为Php7
5.将本目录下的index.php代码替换进在线编译的框架里
6.点击触发器选项，自定义创建触发器
7.触发周期选择自定义触发周期
8.Cron表达式填入：0 0 8 * * * * [即表示每天8点自动运行该函数]
9.点击完成创建即可完成设置
